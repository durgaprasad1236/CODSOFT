This repository contains the projects I completed during my Python Programming Internship at CodSoft.

I created these projects using Python and Tkinter while learning programming concepts, GUI development, and GitHub.

## Projects

### 1. Calculator
A simple console-based calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

It also handles invalid input and division by zero.

---

### 2. Password Generator
A GUI-based password generator built using Python and Tkinter.

Features:
- Generates strong random passwords
- Password strength indicator
- Copy password feature
- Simple and clean interface

---

### 3. Rock Paper Scissors Game
A GUI-based Rock Paper Scissors game using Python and Tkinter.

Features:
- Interactive interface
- Score tracking
- Computer random choices
- Auto reset after each round
- Simple animations and hover effects
